简短笔记一则：查找并删除重复文件

2017年第一则

1, Linux Ubuntu, Mint下， 删除重复文件推荐使用fdupes

安装：

sudo apt-get install fdupes

删除某一个文件夹下的重复文件：

dupes -d -R /some/directory

  1. Windows删除重复文件

推荐 CloneSPY

http://www.clonespy.com/?Home

DoubleKiller
